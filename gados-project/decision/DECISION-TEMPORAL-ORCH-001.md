# DECISION: TEMPORAL ADOPTION FOR BARC ORCHESTRATION

**Decision ID**: DECISION-TEMPORAL-ORCH-001  
**Subject**: Approval to Integrate Temporal for Durable Workflow Execution  
**Authority**: KIW Governance Board  
**Status**: **PENDING REVIEW**

---

## 1. Description
We propose adopting **Temporal** as the primary orchestration engine for the BARC platform. This shift addresses the need for long-running, resilient, and auditable BPO onboarding workflows that can span multiple days and survive system failures.

## 2. Key Benefits
- **Durable State**: No loss of assessment progress during service restarts.
- **Workflow Auditing**: Native support for replaying and inspecting every step of the orchestration.
- **Scalability**: Decouples agent execution from the API request lifecycle.
